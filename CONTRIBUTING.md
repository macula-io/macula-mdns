# Contributing to Macula mDNS

Thank you for your interest in contributing!

## Attribution

This project is a fork of [shortishly/mdns](https://github.com/shortishly/mdns) by Peter Morgan. For significant feature additions or changes to the core functionality, consider contributing upstream first.

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-feature`)
3. Make your changes
4. Run tests (`rebar3 eunit`)
5. Commit your changes (`git commit -am 'Add my feature'`)
6. Push to the branch (`git push origin feature/my-feature`)
7. Create a Pull Request

## Code Style

- Follow existing Erlang conventions
- Add tests for new functionality
- Update documentation as needed
